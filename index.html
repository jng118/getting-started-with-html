Important Concepts from the Intro

Coding

Coding is telling a computer what you want it to do in its own language.


The 5 ways of thinking like a computer programmer

1. Procedural Thinking: is the process of taking problems and breaking it down into a series of steps to address the problem.

2. Abstract Thinking: is the process of using logic to get to the core of an abstract problem, e.g. one apple plus one apple equals two apples. The core logic behind it is 1+1 = 2, even though you are using apples to replace the numbers

3. Systematic Debugging: is the process of identifying problems and tracking it down to its cause to fix the problem

4. Technological Empathy: is understanding how the computer sees things

5. Systems thinking: thinking like a programmer and using the above steps to identify and fix the problem.


The "Interwebs"

The internet is a collection of date. Namely, HTML documents that are contained on servers and accessed by myself or others using a web brower. HTML documents are comprised of text context(what I see), markup (what it looks like), references (other documents like images and videos) and links (links to other pages).


HTML HTML stands for Hyper Text Markup Langauge


Tags and Elements

We use tags to define elements (or what makes up HTML code). For example a tag such as < b > will define that you want the text to contain an element that is "bold". Examples of tags are: < b > - bold, and < em > - italics.


Inline and Blocks


Inline objects are tags you use in line of code to tell the computer how you want that code to display, e.g. using < b > to show that you wanted text in bold. Blocks are tags that you use to put just that, blocks around your text to create form and spacing. You can also modify these blocks by defining its height and width.



Attributes

Attributes are properties of HTML elements. For example, we can use the < href > tag to create an anchor which is basically a link to another document (page, image, etc.) An example of an < href > tag is this: Yahoo!

Syntax You must be careful to write code in a specific way or else the computer will not know what to do with it so that means you have to use tags to define your elements as well as close them to ensure that the computer is doing what you specifically want. For example if I don't use the closing tag on what I want to make bold then the entire paragraph displays in bold.
